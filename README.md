# 今月のシロップ帖

果肉入りシロップのアイデアレシピをスマホで見せるためのPWAです。

## 使い方

ローカルで確認する場合:

```sh
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173/syrup-pwa/` を開きます。

GitHub Pagesでは `https://kitano-sakura.github.io/syrup-recipes/` で公開します。

## 更新しやすい場所

- レシピは `app.js` の `recipes` にあります。
- 今月の見出しと最初に出すレシピは、アプリ内の「今月の表示を編集」から変更できます。
- 画像は `assets/strawberry-syrup-reference.png` です。
